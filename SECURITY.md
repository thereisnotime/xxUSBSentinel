# Security Policy

## Supported Versions

Only the latest release is supported with security fixes.

| Version | Supported |
|---------|-----------|
| latest  | yes       |
| older   | no        |

## Reporting a Vulnerability

Use [GitHub's private vulnerability reporting](https://github.com/thereisnotime/xxUSBSentinel/security/advisories/new) to disclose security issues.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if you have one

You'll receive a response within 7 days. If a fix is warranted, a patched release will be made and the advisory published after a reasonable disclosure window.

Do not open public issues for security vulnerabilities.
